\subsection{Frequency Branch}

\begin{frame}{Why Add a Frequency Branch?}
  \footnotesize
  \begin{itemize}
    \item Generative models (GANs) leave periodic upsampling artifacts — subtle spatially, visible as spectral peaks/asymmetries.
    \item \textbf{Pipeline:} RGB $\to$ FFT2 $\to$ shift $\to$ log-magnitude + phase $\to$ normalize $\to$ 6-channel input $\to$ 4-block CNN $\to$ 256-d feature.
    \item Lightweight by design (no pretrained backbone) — too different from ImageNet textures for transfer learning to help.
  \end{itemize}
